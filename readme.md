### <a id="describe"></a>描述/说明

* 基于lumen封装的一个类restful风格Api Framework.

* 扩展了lumen的路由组件为隐式路由, 不需要再在router.php添加api method.

* 基于百度/淘宝等API规则定义了api auth.

* 扩展了日志 ,Exception日志记录, RequestMiddleware中请求信息记录等.

* ServiceProvider形式扩展redis库.
